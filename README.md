# Arrakis Web Client

This project works as a Web Client for the [Arrakis](http://github.com/blito/arrakis) project.

## Requirements

To build the project, you will need:

* Node.js
* npm
* gulp (npm install -g gulp)

## Build instructions

    git clone http://github.com/blito/arrakis-web.git
    cd arrakis-web
    npm install

To run,

    gulp serve
